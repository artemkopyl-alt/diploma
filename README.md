# 🎬 Cinema Verse — Веб-сайт современного кинотеатра

Интерактивное веб-приложение кинотеатра, выполненное в стиле современных стриминговых платформ, со встроенной логикой выбора сеансов и адаптивным интерфейсом.

---

## 🚀 Инструкция по запуску проекта

Для локального запуска проекта на компьютере не требуется установка дополнительных серверных зависимостей (проект выполнен на стеке HTML/CSS):

1. *Скачивание:* Склонируйте репозиторий или скачайте архив проекта и распакуйте его на рабочий стол.
2. *Запуск:* Перейдите в распакованную папку и дважды кликните по файлу index.html.
3. *Просмотр:* Сайт автоматически откроется в вашем браузере по умолчанию. 
4. *Разработка:* Для просмотра исходного кода откройте папку проекта в редакторе *Visual Studio Code*.

---

## ⚡ Список реализованных функций
* *Каталог фильмов:* Динамическое отображение актуальных фильмов в прокате с интерактивными обложками и возрастным цензом.
* *Интерактивная афиша (afisha.html):* Расписание показов с возможностью интерактивного выбора времени сеанса.
* *Элементы CTA (Call to Action):* Реализованы функциональные кнопки «Билеты» для будущей интеграции со сценариями бронирования.
* *Анимационный фидбек (UX/UI):* Все кликабельные элементы (кнопки, ссылки меню, карточки фильмов) имеют плавный отклик при наведении курсора.
* *Адаптивность:* Полная поддержка отображения интерфейса на мобильных телефонах, планшетах и мониторах высокой четкости.

## 🛠️ Используемые технологии

### 1. Верстка и структура (HTML5)
* Использование строгих *семантических стандартов* (<header>, <nav>, <main>, <section>, <footer>) для обеспечения SEO-оптимизации и правильного чтения структуры поисковыми роботами.
* Применение валидных тегов форм, списков и кнопок <button> для обеспечения интерактивности.

### 2. Стилизация и анимация (CSS3)
* *CSS Grid Layout:* Модульная сетка для отображения каталога фильмов, обеспечивающая автоматическое перестроение блоков (repeat(auto-fit, minmax(...))) без поломок интерфейса.
* *Flexbox Layout:* Позиционирование элементов сквозной шапки сайта и навигационной панели.
* *Микроанимации:* Использование свойств transition и трансформаций transform: translateY для реализации объемного 3D-эффекта карточек при наведении.
* *Адаптивные медиа-запросы (@media):* Оптимизация размеров шрифтов, отступов и сеток под любые разрешения экранов.
